MBNPy

A Python toolkit for white-box risk assessment of complex, large-scale systems

As a companion toolkit of matrix-based Bayesian network (MBN), MBNPy is a tool for risk assessment of large-scale, discrete-state system events.1 MBNPy boldly pursues an innovative pathway to deliver power-saving, fast, and detailed risk solutions.

With large-scale systems, the challenge arises from their giant probability distributions conditioned on numerous components.

MBNPy provides quantification algorithms that encode those giant probability distributions and inference functions that handle encoded distributions.

This is in contrast to recent trends in leveraging High Performance Computing (HPC) power such as Monte Carlo Simulation (MCS) using HPC or training Artificial Neural Network (ANN) to replace an expensive system performance function (i.e. system simulator).

Of course there is no free lunch and MBNPy is no exception. Unlike MCS being applicable to any general systems, MBNPy requires system-specific quantification algorithms. Good news, many system events have already been explored and more research is underway. Check out what system types are available here!

1. For continuous-state events, advanced simulation methods (e.g. subset simulation and Hamiltonian Monte Carlo) have been successfully developed.

  • Multi-objective optimisation: Byun, J. E. & Song, J. (2020). Efficient probabilistic multi-objective optimization of complex systems using matrix-based Bayesian network. Reliability Engineering & System Safety, 200, 106899.
  • Extension to multi-state events: Byun, J. E. & Song, J. (2021). Generalized matrix-based Bayesian network for multi-state systems. Reliability Engineering & System Safety, 211, 107468.
